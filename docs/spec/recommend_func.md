# xkdpi 仕様書（議論機能限定版）

## 1. 概要

**xkdpi** は macOS 用 GUI アプリケーションであり、\
接続されているディスプレイの **HiDPI
表示モードを最適化・適用するツール**である。

本ツールは特に以下の一般的な外部ディスプレイ環境に対応する。

-   **4K (3840×2160)**
-   **WQHD (2560×1440)**

ユーザーの視認性と作業領域のバランスを考慮し、\
**PPI（画素密度）を基に最適な HiDPI UI 解像度を提案し適用する。**

また、設定は保存され、**macOS 再起動後も自動起動して設定を再適用する。**

------------------------------------------------------------------------

# 2. 対象ディスプレイ

本ツールは以下の解像度を主対象とする。

  解像度      説明
  ----------- -------------
  2560×1440   WQHD モニタ
  3840×2160   4K モニタ

理由

-   macOS ユーザーの外部モニタの多くがこの2種類
-   HiDPI スケーリング問題が最も多く発生する領域

------------------------------------------------------------------------

# 3. macOS HiDPI の仕組み

macOS は以下の方式で HiDPI を実現している。

    UI解像度
    ↓
    2倍レンダリング
    ↓
    ディスプレイ解像度へ縮小

例

    UI: 2560×1440
    Render: 5120×2880
    Output: 3840×2160

------------------------------------------------------------------------

# 4. PPI を考慮した最適化

xkdpi は **解像度だけではなく PPI を考慮する。**

PPI 計算式

    PPI = √(width² + height²) / inch

Apple の Retina 設計は

    約 220 PPI

を理想としている。

xkdpi は

    ディスプレイ解像度
    ↓
    PPI計算
    ↓
    最適 UI サイズ計算
    ↓
    HiDPI モード提案

の処理を行う。

------------------------------------------------------------------------

# 5. 推奨 UI 解像度

例

## 27インチ 4K

PPI

    約163

推奨

    2560×1440 HiDPI

理由

-   作業領域と可読性のバランスが良い

------------------------------------------------------------------------

## 32インチ 4K

推奨

    3008×1692 HiDPI

理由

-   文字サイズとUI密度が適切

# 6. 機能

## ディスプレイ検出

取得

-   Display ID
-   解像度
-   リフレッシュレート
-   内蔵 / 外部

------------------------------------------------------------------------

## 表示モード取得

取得

-   UI解像度
-   HiDPI フラグ

------------------------------------------------------------------------

## HiDPI モード識別

表示例

    2560×1440 (HiDPI)

------------------------------------------------------------------------

## 推奨モード表示

GUI 表示

    Recommended
    ⭐ 2560×1440 HiDPI

------------------------------------------------------------------------

## 表示モード切替

ユーザーが GUI からモードを選択し適用できる。

------------------------------------------------------------------------

## 設定保存

保存内容

-   display_id
-   mode_id

------------------------------------------------------------------------

## 起動時復元

macOS 再起動後

1.  アプリ自動起動
2.  設定読み込み
3.  表示モード再適用

------------------------------------------------------------------------
